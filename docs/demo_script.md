# MindBloom Demo Script

1. Open the app and explain that MindBloom supports mood tracking, journaling, habits, reminders, privacy lock, and wellness insight.
2. Go to Settings and show the backend URL plus remote sync status.
3. Create a mood entry with location attached.
4. Create a journal entry with a photo.
5. Create or edit a habit and enable a reminder time.
6. Open the dashboard and show the mood trend, activity signal, and mood map.
7. Go back to Settings, lock the app, then demonstrate passcode or biometric unlock.
8. Explain that the app uses Flutter on the frontend and PHP/MySQL on the backend.
