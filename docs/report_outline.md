# MindBloom Report Notes

## Abstract

MindBloom is a Flutter-based mental wellness application that supports mood tracking, journaling, habit building, reminder scheduling, privacy protection, and location-aware wellness insight. The system combines a mobile frontend with a PHP/MySQL backend for real data storage and synchronization.

## Architecture

- Flutter frontend with Riverpod state management
- PHP API backend for persistent remote storage
- MySQL database accessed through PHP
- local device storage used as resilience and settings cache

## Device Features

- camera for journal photos
- GPS for mood location insight
- sensors for activity signal
- local notifications for reminders
- biometrics and passcode for privacy

## Real-Life Relevance

The system helps users reflect on their emotions, maintain healthy routines, and observe how behavior and place may influence wellness over time.
